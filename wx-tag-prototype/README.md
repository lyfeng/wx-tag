# 微标签小程序原型图

这个文件夹包含了微标签微信小程序的HTML原型图，帮助直观理解小程序的界面和功能流程。

## 原型图内容

1. **首页/登录页** (index.html)：展示微信登录按钮。
2. **我的主页** (home.html)：用户登录后的主页，包含查看自己收到的标签和邀请朋友给自己打标签的按钮。
3. **标签选择页** (tag_selection.html)：展示预设的积极正面标签列表，用户选择标签(最多5个)并确认。
4. **我的标签页** (my_tags.html)：展示用户收到的所有标签及其数量，按数量降序排序，显示AI生成的评语。
5. **管理员标签管理页** (admin.html)：添加、删除、修改预设标签的界面。
6. **直接给好友打标签页** (tag_for_friend.html)：通过分享链接直接进入给特定好友打标签的页面。
7. **标签提交成功页** (tag_success.html)：标签提交成功后的页面，包含"也邀请朋友给我打标签"的按钮。

## 如何使用

1. 直接在浏览器中打开index.html即可开始浏览原型图。
2. 点击界面上的相应按钮可以实现页面之间的跳转，模拟实际应用的交互流程。
3. 在标签选择页可以点击标签进行选择（最多5个）。

## 操作流程

1. **邀请朋友流程**：用户A在主页点击"邀请朋友给我打标签"按钮，生成分享链接发送给朋友或朋友圈。
2. **打标签流程**：用户B收到分享链接后，点击进入直接给用户A打标签的页面(tag_for_friend.html)，选择标签后提交。
3. **连锁邀请流程**：用户B提交标签后，在成功页面(tag_success.html)可以选择"也邀请朋友给我打标签"，形成连锁邀请。

## 技术说明

* 原型图使用HTML、CSS和简单的JavaScript实现。
* 样式基于WeUI设计规范，符合微信小程序的设计风格。
* 原型图仅模拟界面和简单交互，不包含实际的数据存储和处理功能。

## 新增功能（2023年5月更新）

### 1. 我给他人的标签
- 新增了"查看我给他人的标签"功能，用户可以查看自己给其他人打过的标签列表
- 点击列表中的好友，可以查看该好友的标签详情和AI总结
- 支持更新对好友的标签

### 2. 邀请任务列表
- 新增了"我的邀请任务"功能，用户可以查看自己发出的邀请任务列表
- 区分进行中的任务和已完成的任务
- 显示任务的详细信息，包括发起时间、参与人数、剩余时间等
- 支持继续邀请和查看结果操作

### 页面结构
- tagged_by_me.html - 我给他人的标签列表页面
- friend_detail.html - 好友标签详情页面
- invite_tasks.html - 邀请任务列表页面
